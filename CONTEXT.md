# Context

## Glossary

### Template Feature

A template feature is an optional capability that can be enabled or disabled when generating a project.

### Persistence Feature

A persistence feature is a template feature that adds data storage infrastructure to the generated project. Multiple persistence features can coexist in the same generated project.

### Preset

A preset is a named group of template features offered as a convenience. Presets do not restrict which individual features can be selected.
